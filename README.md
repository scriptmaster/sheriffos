#### Pre-requisites

MAC OS X:
```

brew install xorriso
brew install nasm
brew install https://raw.githubusercontent.com/Gallopsled/pwntools-binutils/master/osx/binutils-i386.rb
```

```
brew tap nativeos/i386-elf-toolchain

```
